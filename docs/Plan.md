# WanderCard — 여행 사진 + 텍스트 카드 생성 iOS 앱

> **작성일:** 2026-04-05
> **작성자:** Dani (UX Research)
> **상태:** PoC 개발 시작 (2026-04-05 결정)

---

## 1. 개요

**TravlCard**는 여행 중 찍은 사진과 남긴 메모(장소 정보, 감상, URL 등)를 하나의 **공유하기 예쁜 카드 이미지**로 자동 생성하는 iOS 앱이다.

### 핵심 가치

> "내가 어디 갔는지" 정리가 아니라, **"내가 뭘 느꼈는지" 공유**

Skrap이 여행 사진 정리에 집중한다면, WanderCard은 **사진 + 텍스트 합성 카드**를 통해 여행의 감성과 경험을 시각적으로 기록하고 외부에 공유하는 데 초점을 맞춘다.

---

## 2. 타겟 사용자

- 여행 중 사진을 찍지만 앨범에 쌓만 해두고 정리하지 않는 사람
- 여행 기록을 인스타그램/카카오톡/블로그에 **예쁜 카드**로 공유하고 싶은 사람
- 카카오맵/브라우저에서 발견한 장소를 기록으로 남기고 싶은 여행자

---

## 3. 핵심 기능

### 3.1 사진을 통한 여행 그룹핑

- **PhotosKit**을 통해 EXIF 메타데이터(위치 + 날짜) 기반 여행 단위 자동 그룹핑
  - 예: "2024.06 도쿄" = 6/1~6/7 일본에서 찍은 사진들
- 사용자가 직접 날짜/위치 기반 트리 생성 및 관리

### 3.2 텍스트 수집 — Share Extension 중심 플로우

클립보드 모니터링 대신 **Share Extension**을 메인 진입점으로 삼는다.

**사용자 여정:**
1. 카카오맵, Safari, 인스타그램 등에서 장소/URL/텍스트 복사
2. 시스템 **공유 시트**에서 **"WanderCard로 보내기"** 선택
3. 앱이 데이터를 받아 **카드 미리보기** 생성
4. 원하는 사진과 조합해서 최종 카드 저장/공유

**Share Extension으로 받는 데이터 타입:**
- `URL` — 장소 링크, 블로그 글, 리뷰 페이지
- `Text` — 메모, 인용구, 여행일기 한 줄
- `Image` — 스크린샷, 저장된 사진

### 3.3 카드 생성 엔진

**Core Graphics** 기반 렌더링 파이프라인:

- **라이트 (일기풍)** — 부드러운 배경, 필기체 느낌, 아날로그 감성
- **다크 (감성풍)** — 어두운 배경, 모던 타이포그래피, 밤 감성
- **미니멀 (인스타용)** — 여백 중심, 심플, 인스타그램 스토리 최적화 (1080x1920)

**WCAG 접근성 준수 사항:**
- 카드 텍스트와 배경 간 명도 대비 **AA 기준 (4.5:1)** 충족
- 사진 위에 직접 텍스트를 올리는 레이아웃 → **세미투명 오버레이** 필수
- Dynamic Type 지원 (시스템 폰트 크기 변경 시 카드 내 텍스트도 확대)

### 3.4 공유

- PNG/JPG 내보내기
- iOS 표준 공유 시트 (Instagram, 카카오톡, 메시지 등)

## 4. 기술 스택

| 영역 | 기술 |
|---|---|
| UI | SwiftUI |
| 사진 | PhotosKit (EXIF 메타데이터 기반 필터링) |
| 텍스트 수신 | Share Extension (URL, Text, Image) |
| 카드 렌더링 | Core Graphics (비트맵 합성) |
| 데이터 | SwiftData / CoreData (온디바이스) |
| 서버 | ❌ 없음 (완전 온디바이스) |

### 독서잔디 아키텍처 재사용

샘플 데이터 → 카드 UI → 공유 파이프라인이 이미 구현되어 있어 상당 부분 구조 재활용 가능.

## 7. 온보딩 & 권한 설계

### PhotosKit 권한

**문제:** PhotosKit 접근 거부 시 앱 핵심 기능 불가
**해결 방안:**
1. 권한 요청 전 **"왜 사진이 필요한가요?"** 시각적 설명 화면
2. "여행 사진만 분석합니다. 원본 수정/업로드 없음. 개인정보 저장 안 함." 명확 고지
3. 선택적 접근(Selected Photos)도 부분 지원 — 거부율 최소화

### Share Extension

- 별도 권한 불필요 (시스템 표준)
- 앱 첫 실행 시 "공유 시트에서 찾아보세요" 튜토리얼

## 10. 앱 구조 (정보 아키텍처)

```
WanderCard
├── 홈 (여행 트리 목록)
│   ├── "2024.06 도쿄"
│   ├── "2025.03 제주도"
│   └── + 새 여행 만들기
│
├── 여행 상세
│   ├── 카드 목록 (타임라인 순)
│   ├── + 카드 만들기
│   │   ├── 사진 선택 (PhotosKit 피커)
│   │   └── 텍스트 입력 / Share Extension으로 수신
│   └── 설정 (템플릿 변경, 공유)
│
└── 설정
    ├── 테마 (라이트/다크/미니멀 기본값)
    ├── 사진 접근 권한 관리
    └── 온보딩 다시 보기
```
